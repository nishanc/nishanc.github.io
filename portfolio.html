2020 April | While working as a Software Engineer at 99x

COVID-19 Dashboard for Sri Lanka

COVID-19 Dashboard for Sri Lanka provides the current real time situation of the COVID-19 patients reported in Sri Lanka. Built on top of Angular(Front-end) and Python(Back-end). Used the data provided by Health Promotion Bureau of Sri Lanka. They only provide real time data. I used my own MongoDB database to store the real time data, then used this to visualize historical data as well.
